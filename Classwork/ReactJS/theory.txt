
******React********

    -React is js library developed by Meta to build a fast and interactive user interfaces
    -It is single page application(SPA)

    Page doesnot reload,only content change
    Example-- Insatgram,Facebook,Youtube


Why do we use react/problem with js ?
    Fully page reload , copmlex dom manipulation , hard to manage large application 
    Solution -> updates only require parts to use reusable components


Virtual DOM - 
    It is a lightweight copy of the real DOM used by React to improve performance.
        - when data changes react updates virtual dom 
        - update only changed the part in the dom
        - high performance
