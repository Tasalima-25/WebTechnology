Angular

--type script based frontend framework developed by google for web application
--It is single Page Application-SPA
--It helps us to bulit SPA
--SPA where the page does not fully reload when navigation  

Tools--
node js
npm
Angular cli-command line Interface--used to create and manage angular application

Commands--
1.node -v
2.npm install -g @angular/cli
3.ng new student_management/project_name
4.code .

To run angular project--
    ng serve
    ng s
    ng serve -o

***Component--
    block of code/structure
    UI block of code 
When we create one component-- 
    ng generate component comp_name
    ng g c comp_name

In angular--there are 4 files in folder--
-html
    structure of html code 
-css
    styling of code
-ts
    code of type script
-spec.ts
    testing file



****Package-json---

List packages
include dependencies--production build
dev dependencies--local development
angular.json--project configuration files
style.css--project css style
.gitignore--files or folders to execute from version control
tsconfig.json--typescript configuration files
index.html--main file loads angular 
app.config.ts--app configuration
main.ts--entry point of application
